## Quick Start

The recommended way to get started using [`netlicensing-client`](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22com.labs64.netlicensing%22) in your project is with a dependency management system â€“ the Maven snippet below can be copied and pasted into your build.

Maven:
```xml
<dependencies>
  <dependency>
    <groupId>com.labs64.netlicensing</groupId>
    <artifactId>netlicensing-client</artifactId>
    <version>2.4.1</version>
  </dependency>
</dependencies>
```
Gradle:
```gradle
dependencies {
    compile 'com.labs64.netlicensing:netlicensing-client:2.4.1'
}
```
OSGi:
```
Require-Bundle: com.labs64.netlicensing.client;bundle-version="2.4.1"
```
